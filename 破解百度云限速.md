# 破解百度云限速方法

**Version: unknown**

##Linux&OSX

按照[这里](https://github.com/GangZhuo/BaiduPCS)的方法安装

然后记得设置线程数成5:

>pcs set --max_thread=5

然后登录

>pcs login

程序首先提示输入用户名，我们输入百度网盘的用户名，然后敲入回车。
